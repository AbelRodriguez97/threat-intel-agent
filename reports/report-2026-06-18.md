### 🟠 HIGH — `CVE-2026-54420`

**LiteSpeed cPanel Plugin UNIX Symbolic Link (Symlink) Following Vulnerability** — LiteSpeed / cPanel Plugin

📅 Added to KEV: 2026-06-15 · ⏳ CISA due date: 2026-06-18 · 💀 Ransomware: Unknown

#### Executive summary

The LiteSpeed cPanel Plugin contains a vulnerability that allows for UNIX symbolic link (symlink) following, enabling attackers with existing FTP or web shell access to potentially escalate privileges on shared hosting servers running CloudLinux/CageFS. This vulnerability has been added to the CISA KEV catalog, indicating active exploitation. Organizations should apply vendor-recommended mitigations immediately to prevent potential unauthorized access.

#### Affected technologies

LiteSpeed cPanel Plugin, cPanel, CloudLinux, CageFS

#### Recommended actions

  1. Apply mitigations in accordance with LiteSpeed vendor instructions.
  2. Ensure compliance with CISA’s BOD 26-04 guidance for prioritizing security updates and forensic triage requirements.
  3. For cloud services, follow applicable BOD 26-04 guidance.
  4. If mitigations are unavailable, discontinue use of the LiteSpeed cPanel Plugin.
  5. Evaluate each asset's internet exposure and ensure adherence to CISA BOD 26-04 patching guidelines.

#### Rationale

_This vulnerability is classified as 'high' severity because it is actively exploited, as indicated by its inclusion in the CISA KEV catalog. Although it requires pre-existing FTP or web shell access, it allows for privilege escalation via symbolic link following (CWE-61) in shared hosting environments, which can lead to significant unauthorized access or impact in a multi-tenant setup._
