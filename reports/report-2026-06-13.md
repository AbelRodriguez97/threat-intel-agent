### 🔴 CRITICAL — `CVE-2026-10520`

**Ivanti Sentry OS Command Injection Vulnerability** — Ivanti / Sentry

📅 Added to KEV: 2026-06-11 · ⏳ CISA due date: 2026-06-14 · 💀 Ransomware: Unknown

#### Executive summary

An Ivanti Sentry appliance is vulnerable to an OS command injection, allowing an unauthenticated attacker to remotely execute commands with root privileges. This critical flaw can be exploited if the appliance is unmanaged and externally accessible. Implementing mTLS or restricted HTTPS access can significantly reduce exposure.

#### Affected technologies

Ivanti Sentry, MobileIron Sentry

#### Recommended actions

  1. Immediately apply vendor-provided mitigations and security updates, ensuring compliance with CISA BOD 26-04.
  2. Evaluate all Ivanti Sentry assets for internet exposure, prioritizing those that are unmanaged and externally reachable.
  3. Implement mTLS with EPMM or restrict HTTPS access through Neurons for MDM to make interfaces inaccessible to external actors.
  4. Follow CISA’s “Forensics Triage Requirements” for any potentially compromised systems.
  5. If mitigations are unavailable, discontinue use of the product, following applicable BOD 26-04 guidance for cloud services.

#### Rationale

_This vulnerability allows remote, unauthenticated attackers to achieve root-level code execution on affected Ivanti Sentry appliances. The combination of remote access, lack of authentication, and highest privilege execution makes this a critical security risk, necessitating immediate action._
