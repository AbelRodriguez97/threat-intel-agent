📅 Added to KEV: 2026-06-11 · ⏳ CISA due date: 2026-06-14 · 💀 Ransomware: Unknown

#### Executive summary

A critical vulnerability exists in Ivanti Sentry, allowing remote unauthenticated attackers to execute commands with root privileges. This flaw, an OS command injection, can lead to complete system compromise if the appliance is in an unmanaged state with externally reachable endpoints. Immediate action is required to apply vendor mitigations and secure exposed instances.

#### Affected technologies

Ivanti Sentry, MobileIron Sentry

#### Recommended actions

  1. Apply all available mitigations and security updates from Ivanti in accordance with vendor instructions.

  3. Implement mTLS with EPMM or restrict HTTPS access through Neurons for MDM to limit external interface access.
  4. Ensure compliance with CISA’s BOD 26-04 guidance and “Forensics Triage Requirements”.
  5. Discontinue use of Ivanti Sentry if mitigations are unavailable or cannot be effectively applied.

#### Rationale

_This vulnerability allows for unauthenticated, remote root-level code execution, which is the highest level of compromise. Its inclusion in the CISA KEV catalog signifies active exploitation in the wild, posing an immediate and severe risk to internet-exposed or externally reachable Ivanti Sentry appliances._
