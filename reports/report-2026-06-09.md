This issue is actively exploited and listed in CISA's Known Exploited Vulnerabilities catalog.

#### Affected technologies

Check Point Security Gateway

#### Recommended actions

  1. Immediately apply mitigations as per vendor instructions to all affected Check Point Security Gateway installations.
  2. If vendor-provided mitigations are not available, discontinue the use of the affected product.
  3. Follow applicable CISA BOD 22-01 guidance, particularly for any cloud services integrating with or relying on the affected product.

#### Rationale

The ability to establish a remote access VPN connection without a password provides direct unauthorized access to internal networks, leading to severe impacts on confidentiality, integrity, and availability. Its inclusion in the CISA KEV catalog confirms active exploitation._
